# Headset Battery Widget

Always-on-top, draggable Windows widget that shows the **battery percentage** of supported wireless headsets in real time.

Supported headsets:
- HyperX Cloud II Wireless
- HyperX Cloud II Core Wireless
- HyperX Cloud Stinger 2 Wireless
- HyperX Cloud Alpha Wireless
- Kingston Cloud II Wireless

## Third-party (HIDAPI)
This project uses HIDAPI for USB/HID communication.
